What I do

I build custom web scrapers that extract data from both static and JavaScript-rendered websites, then clean and structure it into a format that's actually usable — no half-finished CSVs with broken rows.

My typical pipeline: Selenium handles the browser and waits for dynamic content to load, BeautifulSoup extracts the data cleanly, and Pandas handles cleaning, deduplication, and export to Excel.

I can also set up scrapers to run on a schedule, so you get fresh data daily or weekly without lifting a finger.
